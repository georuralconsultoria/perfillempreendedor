CONFIGURAÇÃO RÁPIDA

1. Crie um projeto no Supabase.
2. Execute o arquivo supabase-setup.sql no SQL Editor.
3. Copie a Project URL e a Publishable key.
4. Cole os dois valores no arquivo config.js.
5. Abra o site com Live Server e faça um envio de teste.
6. Confira a resposta no Table Editor do Supabase.

IMPORTANTE:
Use apenas a Publishable key no site.
Nunca use Secret key ou service_role key em HTML ou JavaScript público.
